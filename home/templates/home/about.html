About Dashtia

Dashtia is a powerful Kpis tracking and visualization platform.

Features

  • Custom Kpis: Create Kpis for anything you want to track
  • Data Visualization: Beautiful charts and graphs to visualize your data
  • Real-time Updates: See your data update instantly with WebSocket support
  • User-friendly Interface: Clean, responsive design built with Bootstrap

How it Works

  1. Create a new statistic with a descriptive name
  2. Add data items with values and owners
  3. View interactive charts and analytics
  4. Track changes over time
